Given Arduino R4 WiFi board, a WS2812B led strip with 3x28 RGB LEDs, 5V 3A power adapter, IR receiver and 24-key IR remote.

The project is to create a floor column lamp with possibility to manage the LED strip using the Arduino-hosted web application or mobile application using Bluetooth connection.

Also as another way of working with the lamp, connect the IR receiver to the Arduino board and learn the IR remote's commands to operate the IR strip.

Files with details about the hardware:
ABX00087-full-pinout.pdf - Arduino Uno R4 WiFi pinout
WS2812B-datasheet.pdf - addressable RGB LED datasheet
TSOP-1838 Sensor.pdf - IR receiver datasheet
